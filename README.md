##Review AI Agent

An AI-powered application that automatically generates responses to customer reviews.

##Features

Automatically generates responses to customer reviews.

Detects sentiment of the review (Positive / Negative).

Sends an alert email to the support team if a review is negative.

Stores review details in a database for tracking and analytics.

##Workflow

User submits a review through the interface.

The AI agent analyzes the sentiment.

A response is generated automatically.

If the review is negative, an alert email is sent to the support team.

All review details are stored in the database.

##Tech Stack

Python

Streamlit

FastAPI

LangGraph / LLM

SQLite Database

review_ai_agent/

##Purpose

This project demonstrates how an AI agent can automate customer feedback handling by generating responses, detecting negative sentiment, notifying support teams, and maintaining a database of reviews.
